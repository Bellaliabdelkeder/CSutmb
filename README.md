## How to Contribute

# 🚀 How to Contribute (Simple Way)

1.  Fork the repository\
2.  Add your file in the correct folder\
3.  Respect the naming convention\
4.  Submit a Pull Request

------------------------------------------------------------------------

# 👨‍💻 Professional Contribution Guide (Git Users)

## 1️⃣ Clone Your Fork

``` bash
git clone https://github.com/Alialalaa/CSutmb
cd CSutmb
```

## 2️⃣ Add Original Repository as Upstream (Important)

``` bash
git remote add upstream https://github.com/OWNER_USERNAME/CSutmb.git
```

This keeps your fork updated with the main project.

## 3️⃣ Create a New Branch

``` bash
git checkout -b example-add-math-exam-2024
```

## 4️⃣ Add Your Files

``` bash
git add .
git commit -m "Add math exam 2024"
git push origin example-add-math-exam-2024
```

------------------------------------------------------------------------

# 🔄 Create a Pull Request

1.  Go to your fork on GitHub\
2.  Click **Compare & Pull Request**\
3.  Write a short description\
4.  Click **Create Pull Request**

------------------------------------------------------------------------

# 🛠 Keeping Your Fork Updated

``` bash
git fetch upstream
git checkout main
git merge upstream/main
git push origin main
```
